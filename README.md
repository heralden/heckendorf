# heckendorf

Procedurally generated roguelike dungeon crawler game implemented as a fullstack Clojure webapp.

Live at: http://heck.8620.cx

# Running locally

```
lein start
```

# Development

## Server

```
lein repl
heckendorf.web=> (-main)
```

Or `lein run` if you haven't been REPLightened yet.

## Client

```
lein figwheel dev
```
